公司简介

浙江诚邦园林规划设计院有限公司（以下简称“诚邦设计”）成立于2003年11月，是浙江诚邦园林股份有限公司（股票代码：603316）的全资子公司。

诚邦设计拥有风景园林工程设计专项甲级、城乡规划乙级、绿化造林设计丙级等资质，获得ISO9001质量、环境、职业健康安全管理体系认证，现有各类设计人员百余人，下设深圳、福州、西安等分院，提供城乡规划、小城镇创新、特色产业、居住区、城市绿地、综合体等多领域的设计、PPP及EPC多模式的服务工作，现已在浙江、上海、深圳、陕西、江西、山西、山东、河南、江苏、内蒙古等多个省市成功开展设计业务。

诚邦设计秉承“以德立人、以诚兴邦”的企业文化，坚持并致力于成为国内一流的设计服务商，将一如既往的为客户提供质量卓越、理念新颖的设计作品，为城市园林景观的发展做出更大的贡献。
